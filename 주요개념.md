# 데이터 유형별 분석 구분

| 구분          | 범주형 데이터 (Categorical)                                | 비범주형 데이터 (Continuous)       |
|-------------|------------------------------------------------------|-----------------------------|
| **의미**      | 구분, 등급 등으로 나뉘는 분류형 자료                                | 연속적인 수치로 표현되는 자료            |
| **예시**      | 연료등급(1~5등급), 성별, 지역, 수능등급                            | 연료소비량(L), 점수, 키, 몸무게, 아파트평수 |
| **주요 분석방법** | 분류(Classification), 카이제곱검정, 로지스틱회귀, 랜덤포레스트, 그라디언트부스팅 | 회귀(Regression), 선형회귀, 다중회귀  |
| **핵심관점**    | ‘어느 그룹인가?’ (빈도·비율 중심)                                | ‘얼마나인가?’ (평균·변화량 중심)        |

---

### 💡 한 줄 요약

데이터는 **범주형(분류 중심)** 과 **비범주형(회귀 중심)** 으로 나뉘며,  
데이터의 **형태에 따라 분석방법이 결정**된다.

# 선형회귀 vs 로지스틱 회귀 비교 요약

| 구분          | 선형회귀 (Linear Regression) | 로지스틱 회귀 (Logistic Regression)          |
|-------------|--------------------------|----------------------------------------|
| **목적**      | 연속값 예측 (예: 집값, 점수)       | 분류(Classification) (예: 합격/불합격)         |
| **수식 형태**   | 𝑦 = a₁x₁ + a₂x₂ + … + b | σ(z) = 1 / (1 + e⁻ᶻ), z = a₁x₁ + … + b |
| **출력값 범위**  | (-∞, +∞)                 | (0, 1) 확률로 해석                          |
| **결과 해석**   | 예측값 자체가 의미               | 확률을 기준으로 0/1로 분류                       |
| **대표 손실함수** | MSE (Mean Squared Error) | Log Loss (Binary Cross Entropy)        |
| **비유**      | “시간 늘면 점수도 선형적으로 증가”     | “점수가 높을수록 합격 확률이 S자 형태로 증가”            |

---

## 간단 예시

- **선형회귀:**  
  공부시간(x) → 점수(y)  
  예: `y = 10x + 40`  
  → 공부시간이 늘면 점수도 비례 증가

- **로지스틱 회귀:**  
  공부시간(x) → 합격여부(y=0/1)  
  예: `p(합격) = σ(0.8x - 2.5)`  
  → 공부시간이 많을수록 합격 확률이 점점 높아짐
